osu!（下面简称osu）是一款历史悠久的免费音游。它玩起来看着像在“戳泡泡”，曲目超多，不停有创作者在创作新曲的面谱供大家免费下载。

## 安装

进入[osu官网](https://osu.ppy.sh/)，根据指引就可以下载到osu下载器。下载器可能下载比较慢，如果有条件可以尝试科学上网，耐心等待。

## 账号

osu的理念是一人一号。请起一个好听的昵称，改名是需要花钱的，而且越来越贵，第一次就要5美元！它不鼓励改名。共享账号，开小号都有封号的可能！

## 谱面

找到自己感兴趣的曲子，是保持对游戏兴趣的一大要点。曲子可以是你喜欢的番剧的op、ed，也可以是玩的游戏的主题曲、BGM，或者偶像歌手、流行曲等等。我建议去B站等社区找其他人整理的曲包，只要有一个id列表就可以了，拿着id去官网上下载。另一个获取新曲子的方法是玩多人游戏，多人游戏在房间中房主会选图，大家都要下载这张图才能参与游戏。往往玩家选的图都是质量比较高的图或者是大家喜闻乐见的图。更直接的方法是，直接去官网搜你喜欢的曲子。

## 设备与手法

根据使用设备划分，有键鼠玩家、纯鼠玩家、数位板玩家等。我不想直接指定你说你要使用什么设备，练习，熟悉，进步，遇到瓶颈，改变，练习，这是一个循序渐进的过程，也是这个游戏的一个乐趣来源。

### 新手-纯鼠

作为一个新手，你可以先只用鼠标试一试新手关卡。鼠标移动来带动光标移动，到时间点就点击鼠标左键。

如果你是一个FPS玩家你大概率会知道，像windows电脑默认的鼠标移动，并不是和你的手线性映射的，而是你手移动的越快，电脑光标的加速度就越大！为了提高精度，你需要进入设置->鼠标设置->其他鼠标设置->指针选项->提高指针精确度（相反它根本提高不了精准度），把这一项关掉！这样就变成了光标速度与手的速度成正比，这对于osu来说这也是几乎必要的。

### 加入键盘

但逐渐地你可能会发现一只手有些忙不过来，你可以用敲击键盘来代替点击鼠标，默认是z键和x键，它们是一样的，你可以先只用一个按键。左手用键盘进行敲击，右手用鼠标控制光标。

接着你可能遇到了二连打，一个键连续按两下有些困难，你可能需要用上左右两根手指，连续地按下xz，这样就实现了连打。相应的，三连打就是xzx，以此类推。

### 数位板

想要提高定位的精准度，从鼠标换成数位板将是一个好的选择。像Wacom CTL-472这种小尺寸数位板是比较合适的。买来数位板之后记得在数位板驱动中将映射改为绝对映射。

有人（比如我）刚买来数位板发现这东西真神奇，笔不用触碰在板子上隔空就可以控制光标。于是我想使用纯数位板来打osu。但是玩了几天发现在笔不触碰板子的时候定位是不可控的，而且用笔敲击板子来打击会跟不上节奏。

正确的做法是：在osu中禁用鼠标左键，游玩过程中笔一直按在板子上划线，敲击还是使用键盘来进行。当你熟练之后你会发现你能打难度更高的图了！

### 单双与强双

单双和强双是左手敲击键盘的两种手法。使用固定的手法可以让你的成绩稳定，应对各种连击。单双指的是，除了连打，一直用一根手指敲击，遇到连打才使用第二根手指，例如：x x x xzx x x x xzxzxzx。我一开始是单双，但是当我想打四星图的时候就发现我遇到瓶颈了，不是连打跟不上，而是一些节奏快的单气泡我跟不上，于是我去练了强双。

强双指的是，不管任何情况，两根手指交替使用。完美的强双打完一个曲子，你的z和x的敲击次数应该是相等或仅相差1次的。一开始练感觉这很难适应，甚至有点反人类。后来看了一个教学视频，他提到一点“强双的触发时机不是按下的那一刻，而是双指交换的那一刻”。这个很关键，进行这一个思想转变，我很快就练到了原来单双的水平。

强双还分为抬指强双和不抬指强双。我只会不抬指强双，这方面已经触及到我的水平上限了。下面引用一位贴吧老哥的回复，他讲的很好。先练不抬指强双，不抬指强双不用考虑按哪根手指，不用考虑是单点还是滑条还有“切指”的概念，这几点我是非常认同的。
