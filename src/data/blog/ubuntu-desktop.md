## 窗口操作

| 快捷键 | 功能 |
|--------|------|
| `Alt + F4` | 关闭窗口 |
| `Alt + F10` | 最大化窗口 |
| `Alt + F9` | 最小化窗口 |
| `Alt + Tab` | 切换应用程序 |
| `Alt + `` | 在同一程序的窗口间切换 |
| `Super` | 打开应用程序菜单 |

## 工作区

| 快捷键 | 功能 |
|--------|------|
| `Super + S` | 打开工作区概览 |
| `Super + ←/↑/↓/→` | 移动窗口到不同工作区 |
| `Ctrl + Alt + ←/→` | 切换工作区 |

## 截图

| 快捷键 | 功能 |
|--------|------|
| `Print` | 截取整个屏幕 |
| `Alt + Print` | 截取当前窗口 |
| `Shift + Print` | 截取选定区域 |

## 文件管理器 (Nautilus)

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + N` | 新建窗口 |
| `Ctrl + T` | 新建标签页 |
| `Ctrl + W` | 关闭标签页 |
| `Ctrl + L` | 跳转到地址栏 |
| `Ctrl + H` | 显示/隐藏隐藏文件 |
| `F2` | 重命名 |
| `Delete` | 移到回收站 |

## 终端

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + Shift + T` | 新建终端标签页 |
| `Ctrl + Shift + W` | 关闭终端标签页 |
| `Ctrl + Shift + C` | 复制 |
| `Ctrl + Shift + V` | 粘贴 |
| `Ctrl + D` | 退出当前终端 |
| `Ctrl + R` | 搜索历史命令 |

## 系统

| 快捷键 | 功能 |
|--------|------|
| `Super` | 打开Activities搜索 |
| `Alt + F1` | 聚焦到Applications |
| `Super + D` | 显示桌面 |
| `Super + L` | 锁屏 |
| `Ctrl + Alt + Delete` | 登出 |
